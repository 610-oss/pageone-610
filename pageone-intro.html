👣 我们的四周旅程

Week 1：拥有你的网站

注册 GitHub → Fork 模板 → 启用 GitHub Pages → 改第一行文字。 让每个人都能看见自己上线的第一张网页。

Week 2：做成你的样子

用最简单的方式，让页面更像自己：改颜色、换字体、放上喜欢的图。 不需要学 CSS 语法——用 AI 做你的编程小助手。

Week 3：让网页活起来

接入 JSON / Supabase，初步感受“数据 + 页面”的魔力。 让网页不仅是死图，而是你自己的小工具、小应用。

Week 4：展示你的成果

给你的作品起个名字，配上介绍，截图分享到群里 / 朋友圈。 这是你亲手完成的第一张数字作品，也许是更多创造的起点。

✨ 你将获得

  • 一个上线的个人静态网站（可自定义）
  • 最简单的网页编辑 & 样式修改能力
  • 数据接入 & AI 协作的体验
  • 一个公开分享的作品
